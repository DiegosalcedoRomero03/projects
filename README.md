# Sitio de muestra — Producto con Modelos

Plantilla simple y moderna para presentar un producto que posee varios modelos. Está pensada para que añadas fotos, descripciones y precios más tarde.

Instrucciones rápidas:

- Abrir en el navegador: abre `index.html` en tu navegador (doble clic o arrastrar al navegador).
- Añadir imágenes: coloca imágenes en una carpeta `assets/` dentro de `mi-tienda` y en el HTML agrega la ruta en el atributo `data-image` del `article.model`, por ejemplo `data-image="assets/modeloA.jpg"`.
- Actualizar nombre/descr: edita los atributos `data-name` y `data-desc` en cada `article.model` o modifica el texto dentro de la tarjeta.
- Añadir precio: agrega un nuevo elemento en la tarjeta y/o en el panel de detalles y estilízalo en `styles.css`.

Estructura creada:

- `mi-tienda/index.html` — entrada principal con secciones y placeholders para modelos.
- `mi-tienda/styles.css` — estilos modernos, variables CSS y diseño responsivo.
- `mi-tienda/script.js` — lógica para seleccionar modelos y mostrar detalles.
- `mi-tienda/README.md` — instrucciones y próximos pasos.
